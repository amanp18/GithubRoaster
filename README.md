## 🚀 What is GitHub Roaster?

**GitHub Roaster** is a web-based tool (or CLI/API-based, depending on your build) that generates **intelligent, savage, and soul-piercing roasts** targeting ego, confidence, and existence — without using profanity. It’s sarcasm, wrapped in wit, dipped in psychological fire.

Perfect for:
- Testing emotional reactions  
- Roasting friends (or enemies)  
- Content creators & streamers  
- Personal reality checks  
- Just... chaos.

---

## 🧠 Features

- 🔥 Devil-mode roasting (Lucifer / Grok style)
- 🎯 Emotion-based targeting: Ego, intelligence, confidence
- 🧊 Cold, sarcastic tone with zero mercy
- 🧪 Experiment mode: Analyze how fast the user gets triggered
- 🛡️ Profanity-free but deeply brutal (optional explicit toggle)
- 🌍 Hindi + English support
